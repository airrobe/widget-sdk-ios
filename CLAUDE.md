## Overview

Native iOS SDK providing AirRobe widgets (OptIn, MultiOptIn, Confirmation) for integration into iOS apps. Distributed via Swift Package Manager.

## Tech Stack

- Swift 5.3+, iOS 13.0+, Xcode 12.0+
- Swift Package Manager

## Commands

- Build and test via Xcode

## Architecture

- Widget views: AirRobeOptIn, AirRobeMultiOptIn, AirRobeConfirmation
- Distributed as a Swift Package
